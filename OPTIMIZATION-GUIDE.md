# 🚀 Panduan Optimasi Gambar untuk Website Album

## ⚠️ PENTING: Langkah ini akan membuat website jauh lebih cepat!

Website Anda saat ini lambat kemungkinan karena gambar JPG yang besar (mungkin beberapa MB per gambar). 
Ikuti panduan ini untuk mengoptimalkan gambar.

---

## ✅ Cara Mudah: Menggunakan Tool Online Gratis

### Opsi 1: TinyPNG (Paling Mudah)
1. Buka: https://tinypng.com/
2. Upload semua file .JPG dari folder Anda
3. Tunggu proses kompresi selesai
4. Download gambar yang sudah dikompres
5. Ganti file lama dengan file baru di folder album

**Hasil:** Gambar bisa turun 50-80% lebih kecil tanpa kehilangan kualitas!

### Opsi 2: Squoosh (Lebih Banyak Opsi)
1. Buka: https://squoosh.app/
2. Drag & drop gambar Anda
3. Pilih format **WebP** (lebih baik dari JPG)
4. Download hasil optimasi
5. Ganti file lama

---

## 🎯 Target Ukuran Gambar

| Jenis Gambar | Ukuran Ideal | Format |
|--------------|--------------|--------|
| Foto Member (portrait) | 100-300 KB | JPG atau WebP |
| Foto Album | 150-400 KB | JPG atau WebP |
| Background Hero | 200-500 KB | JPG atau WebP |
| Logo/Icon | 10-50 KB | PNG atau SVG |

---

## 📋 Langkah-langkah Detail

### Step 1: Kompres Semua Gambar
1. Install aplikasi gratis: **RIOT** (Windows) atau gunakan online tools
2. Kompres semua file .JPG dengan kualitas 70-80%
3. Resize ke ukuran maksimal: **1920x1080 pixels**

### Step 2: Resize Gambar (Jika Perlu)
- Foto profil member: **800x800 pixels** sudah cukup
- Foto album: **1200x1200 pixels** maksimal
- Background hero: **1920x1080 pixels**

### Step 3: Pertimbangkan Format WebP
WebP lebih kecil 25-35% dari JPG dengan kualitas sama.
Convert menggunakan: https://cloudconvert.com/jpg-to-webp

## 💡 Tips Tambahan

1. **Jangan gunakan gambar > 1MB** - Ini penyebab utama website lambat
2. **Gunakan JPG untuk foto** - Lebih baik dari PNG untuk foto
3. **Hindari PNG** - Kecuali untuk logo/transparansi
4. **Kompres sebelum upload** - Selalu kompres gambar dulu
5. **Test kecepatan** - https://pagespeed.web.dev/

---

## 📊 Estimasi Peningkatan Kecepatan

| Sebelum | Sesudah |
|---------|---------|
| 5-10 detik loading | 1-2 detik loading |
| 50MB+ total gambar | 5-10MB total gambar |
| Lag saat scroll | Smooth scroll |
